`DateInputGroup` is a presentational shell for date text entry and segmented date display. It does not parse, validate, or store date values.

## Import

```tsx
import { DateInputGroup } from "@pitsi-ui/native/date-input-group";
```

## API Reference

### DateInputGroup Slots

| Slot | Description |
| --- | --- |
| `DateInputGroup.Root` | Root input frame. |
| `DateInputGroup.InputContainer` | Flexible content area. |
| `DateInputGroup.Input` | TextInput configured for date punctuation. |
| `DateInputGroup.Segment` | Static date segment text. |
| `DateInputGroup.Prefix` | Leading slot. |
| `DateInputGroup.Suffix` | Trailing slot. |
